# Excellent ユーザー権限設計
- 管理者ユーザー  
  一旦は用意しない。ユーザーが登録したマシン, ユーザーを削除するなどの管理者用コマンドを用意する

- ログインユーザー
  - バッティングセンター一覧画面
    - バッティングセンターの検索
    - バッティングセンターの検索結果の一覧表示
    - 行った！数の閲覧
    - 行った！ボタンの使用
    - バッティングセンター詳細画面への遷移
  - バッティングセンター詳細画面
    - 行った！数の閲覧
    - バッティングマシーン情報の閲覧
    - あった！なかった！数の閲覧
    - バッティングマシーン情報の投稿
    - 行った！ボタンの使用
    - あった！なかった！ボタンの使用
  - マイページ画面
    - 行った！を押したバッティングセンターの一覧表示
    - 投稿したマシーン情報の閲覧
    - 投稿したマシーン情報の編集
    - 投稿したマシーン情報の削除

- 非ログインユーザー
  - バッティングセンター一覧画面
    - バッティングセンターの検索
    - バッティングセンターの検索結果の一覧表示
    - 行った！数の閲覧
    - バッティングセンター詳細画面への遷移
  - バッティングセンター詳細画面
    - 行った！数の閲覧
    - バッティングマシーン情報の閲覧
    - あった！なかった！数の閲覧
